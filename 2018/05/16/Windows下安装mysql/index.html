Windows下安装mysql

发表于 2018-05-16

最近工作中需要用到数据库mysql，于是就来一场mysql之旅。下面总结下在Windows下安装mysql过程（Windows7 64位）

1.下载安装包

打开mysql官网下载页：mysql官网下载页

选择相对应的版本和平台

下载解压之后放到目录中，本教程统一放在D:\mysql-8.0.11

2.mysql配置

在网上看的教程，都需要配置my-default.ini文件，我这下载解压后的没有这个配置文件，所以就没配置，执行后边的步骤也能正常使用mysql

3.安装启动mysql

  • 用管理员身份打开cmd命令行工具，cd到上边解压文件的bin目录下：D:\mysql-8.0.11\mysql-8.0.11-winx64\bin
  • 输入mysqld install 回车运行，出现安装成功即可
  • 输入net start mysql 启动mysql

在cmd里发现无法启动

输入一下命令自动生成无密码的root用户

mysald --initialize-insecure

初始化后再运行net start mysql成功启动mysql

4.进入mysql环境

可以使用以下命令来创建root用户的密码：

mysqladmin -u root password "new_password"

然后通过以下命令来连接mysql服务器：

mysql -u root -p

会提示输入密码，输入密码之后即可进入mysql环境

如果退出mysql，输入exit回车既可以

5.执行mysql语句 show databases；即可看到默认的数据库

6.配置sql环境变量

不想每次cd到mysql的bin目录下执行命令，可以配置环境变量，然后就可以随时随地使用sql命令。

配置方法：右键我的电脑–属性–高级系统设置–环境变量–path–编辑，将mysql的bin的目录放里面：D:\mysql-8.0.11\mysql-8.0.11-winx64\bin

# mysql
