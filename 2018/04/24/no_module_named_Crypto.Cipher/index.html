Python报错no module named Crypto.Cipher

发表于 2018-04-24

Windows在运行Python项目时，报错no module named Crypto.Cipher刚开始以为是没有这个库，用常规的pip install crypto。安装完成之后，依然报错

在网上搜索结果是需要安装pycrypto，首先卸载之前安装的crypto，然后安装pycrypto，用如下方法：

pip uninstall Crypto
pip uninstall pycrypto
pip install pycryp

在此过程中安装pycrypto失败，解决办法：

在http://www.voidspace.org.uk/python/modules.shtml#pycrypto直接下载编译好的源文件。运行，安装，成功，之后就可以愉快的使用crypto.
参考链接：http://stackoverflow.com/questions/11405549/how-do-i-install-pycrypto-on-windows
