no module named Crypto.Cipher

发表于 2018-04-24

Windows在运行Python项目时，报错no module named Crypto.Cipher刚开始以为是没有这个库，用常规的pip install crypto。安装完成之后，依然报错

在网上搜索结果是需要安装pycrypto，首先卸载之前安装的crypto，然后安装pycrypto，用如下方法：

pip uninstall Crypto
pip uninstall pycrypto
pip install pycryp

在此过程中安装pycrypto失败，解决办法：

在http://www.voidspace.org.uk/python/modules.shtml#pycrypto直接下载编译好的源文件。运行，安装，成功，之后就可以愉快的使用crypto.
参考链接：http://stackoverflow.com/questions/11405549/how-do-i-install-pycrypto-on-windows

Python遍历文件夹中的文件

发表于 2018-04-24

最经刚开始接触Python，学习了用Python实现遍历文件夹里的文件（只遍历文件不要目录），并将这些文件放到一个列表中

废话不多说，直接上代码

def allDir(path):
    f=[]
    for root,dirs,files in os.walk(path):
        for filespath in files:
            print(os.path.join(root,filespath))
            f.append(os.path.join(root,filespath))

    print(f)
    return(f)

错误提示：Unable to connect to adb.Check if adb is installed correctly

发表于 2018-04-24

stackoverflow同问题，这个链接help我解决此问题

最近刚开始使用autoIT，需要通过uiautomatorviewer.bat（Android sdk中）来找到屏幕上控件的ID或者text，用来编写相应脚本

从官网下载Android sdk，找到对应的uiautomatorviewer.bat，打开发现出现错误提示：Unable to connect to adb.Check if adb is installed correctly

通过网上搜索相关内容，得以解决，方法如下：

用记事本打开uiautomatorviewer.bat，找到call %java_exe% -Djava.ext.dirs=%javaextdirs% -Dcom.android.uiautomator.bindir= -jar %jarpath% %

把它改为call "%java_exe%" "-Djava.ext.dirs=%javaextdirs%" "-Dcom.android.uiautomator.bindir=D:\Android\tools" -jar %jarpath% %*

然后copy adb.exe 到bin 所在路径。再次运行，不再提示错误信息

Win7自带压缩和解压zip功能

发表于 2018-04-24

最近用widows电脑需要将文件夹压缩成zip文件，查询方法如下：

1、使用Windows7压缩zip文件

鼠标右键单击要压缩的文件夹或者选择的多个文件，选择“发送到 -> 压缩(zipped)文件夹”，然后系统就自动压缩成一个与文件夹同名的zip压缩文件。

2、使用Windows7解压缩zip文件

解压缩zip文件也很简单，有两种方法：

1、鼠标右键zip压缩文件，选择“全部提取”，然后按照步骤提取到某一个目录就可以了。

2、如果你的鼠标右键没有“全部提取”这一选项，就可以使用这个方法了。标右键zip压缩文件，选择“打开方式 -> Windows资源管理器”，然后在左上角可以点击“提取所有文件”。或者就当这个压缩包为一个文件夹，在资源管理器中复制&拷贝也可以。

测试时一些手机不能打印日志

发表于 2018-04-24

测试客户端APP时，需要打印些日志来测试以及定位bug，但是在测试过程中总有一些手机不按套路出牌（比如一些手机默认不能打印log需要手动设置才能打印出log日志）

魅族pr5设置方法：打开【设置 】中的【开发者选项】，页面底部找到【性能优化】，打开【高级日志输出】勾选【全部允许】即可。

华为手机：不是在【开发者选项】中，而是打开拨号界面的拨号盘，输入##2846579##，系统会自动打开【工程菜单】界面，依次打开【后台设置】->【log设置】，勾选【AP日志】即可。
